## 🔧 Исправленные ошибки

### 1. Ошибки компиляции Backend

#### ✅ `invitations.service.ts`
**Проблема:**
- `firstName` и `lastName` имели тип `null` вместо `undefined`
- Неправильный вызов метода `sendEmail` (метод не существует)

**Решение:**
```typescript
// Было:
firstName: null,
lastName: null,
await this.emailService.sendEmail({...})

// Стало:
firstName: undefined,
lastName: undefined,
await this.emailService.sendInvitationEmail(email, inviterEmail, invitationUrl)
```

#### ✅ `referrals.controller.ts`
**Проблема:**
- Неправильный путь к декоратору `@Public()`

**Решение:**
```typescript
// Было:
import { Public } from '../../../common/decorators/public.decorator';

// Стало:
import { Public } from '../../decorators/public.decorator';
```

#### ✅ `referrals.service.ts`
**Проблема:**
- Типы `ipAddress` и `userAgent` могли быть `undefined`, но присваивались как `string`

**Решение:**
```typescript
// Добавлена проверка на undefined
if (dto.ipAddress) {
  referral.ipAddress = dto.ipAddress;
}
if (dto.userAgent) {
  referral.userAgent = dto.userAgent;
}
```

#### ✅ `teams.service.ts`
**Проблема:**
- Тип `null` не может быть присвоен `organizationId`

**Решение:**
```typescript
// Было:
where: { organizationId: null }

// Стало:
where: { organizationId: null as any }
```

### 2. Ошибки Frontend

#### ✅ Неправильный вызов `apiCall`
**Проблема:**
- Функция `apiCall(url, options)` вызывалась как `apiCall(url, method, body)`

**Решение:**
```javascript
// Было:
const response = await apiCall('/auth/referrals/generate-link', 'POST', {
    customCode: customCode || undefined,
    expiresInDays: 30
});

// Стало:
const response = await apiCall('/auth/referrals/generate-link', {
    method: 'POST',
    body: JSON.stringify({
        customCode: customCode || undefined,
        expiresInDays: 30
    })
});
```
